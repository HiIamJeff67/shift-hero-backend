# Shared GraphQL Resources

This directory contains shared GraphQL schema and query assets used by the Notezy backend template.

## Notes

- These files are maintained as a reusable GraphQL foundation.
- The backend architecture is developed and fully owned by Notezy.
- This repository is published under Apache-2.0 by Notezy.

## Structure

- `schemas/`: SDL schema files
- `schemas/enums/`: enum definitions
- `fragments/`: reusable query fragments
- `queries/`: reusable query templates
